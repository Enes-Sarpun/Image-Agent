## 0. BAĞLAM (Önce Oku)

Tek frontend var: `index.html`. `app.py` (Gradio) SİLİNECEK — onunla işin yok.
Backend: `api.py` (FastAPI), şu endpoint'ler aktif:
  • POST /analyze   → multipart form, "image" alanı, JSON döner:
      {
        "verdict": "ai" | "real",
        "confidence": 0-100 float,
        "reasoning": str,
        "key_indicators": str[],
        "source": "watermark" | "llm" | "llm+forensic" | "multi_pass+forensic"
                  | "llm_search+forensic" | "cache",
        "elapsed_ms": float
      }
  • GET /health     → {"status":"ok"}

Yani plan = tek bir güçlü `index.html` + vanilla JS (+ tercihen `style.css` ve
`script.js` ayrı dosyalara taşınmış). Build tool YOK, framework YOK.
Sadece statik dosyalar; FastAPI bunları StaticFiles ile serve edecek.

Mevcut index.html'in sorunları:
  • Tanıtım sayfası olarak duruyor — upload/analiz hiç YOK, sadece "GitHub'da
    incele" diyor. Halbuki ürünün kendisi tarayıcıda çalışmalı.
  • Hero güzel (Cormorant italic + grayscale), bu yön korunacak.
  • Features ve pipeline bölümleri uzun, ana iş akışından önce geliyor.
  • Tek bir glow blob var — bu temiz, devam.
  • Marquee (tech-track) var ve KALDIRILACAK — dikkat dağıtıyor.

### 5.2 Fonksiyonlar

  • initApp()              → DOMContentLoaded'da çağrılır
  • initDropzone()         → drag/drop/click handler'lar
  • handleFileSelect(file) → File validation (boyut/format), preview göster
  • clearFile()            → currentFile = null, dropzone'u sıfırla
  • analyzeImage()         → fetch POST /analyze, loading state, sonuç render
  • renderVerdict(data)    → verdict card HTML'ini güncelle, stagger animasyon
  • renderEmpty()          → empty state'i göster
  • showSkeleton()         → loading skeleton göster
  • showToast(msg, type)   → top-right toast (success/error)
  • countUp(el, target)    → 0'dan target'a sayı animasyonu (requestAnimationFrame)
  • initReveal()           → IntersectionObserver scroll reveal
  • initParallax()         → hero parallax (mevcut kod)

### 5.3 API Çağrısı

  async function analyzeImage() {
    if (!currentFile || isAnalyzing) return;
    isAnalyzing = true;
    setButtonLoading(true);
    showSkeleton();

    const formData = new FormData();
    formData.append('image', currentFile);

    try {
      const res = await fetch(`${API_BASE}/analyze`, {
        method: 'POST',
        body: formData,
      });
      if (!res.ok) {
        const err = await res.json().catch(() => ({ detail: 'Bilinmeyen hata' }));
        throw new Error(err.detail || `HTTP ${res.status}`);
      }
      const data = await res.json();
      renderVerdict(data);
    } catch (err) {
      showToast(err.message, 'error');
      renderEmpty();
    } finally {
      isAnalyzing = false;
      setButtonLoading(false);
    }
  }

NOT: Cache checkbox'ı şu an backend tarafında API parametresi olarak
yok — `agent.analyze(use_cache=...)` kwarg'ı var ama `/analyze` endpoint'i

ÖNERİ: (B) seçeneği — api.py'ye 1 satırlık ekleme, çok kolay. Sonnet bunu

## 9. BACKEND ENTEGRASYONU
═══════════════════════════════════════════════════════════════════════════════

FastAPI (`api.py`) iki ekleme yapmalı:

  1. StaticFiles ile index.html serve etmek:
     ```python
     from fastapi.staticfiles import StaticFiles
     app.mount("/", StaticFiles(directory=".", html=True), name="static")
     ```
     ÖNEMLİ: Bu mount EN SONA eklensin, /analyze ve /health route'larından
     SONRA — yoksa onları gölgeler.

  2. CORS gerekirse (aynı origin'den serve ediliyorsa gerekmez, sadece ayrı
     port'tan dev sırasında):
     ```python
     from fastapi.middleware.cors import CORSMiddleware
     app.add_middleware(CORSMiddleware, allow_origins=["*"],
                        allow_methods=["*"], allow_headers=["*"])
     ```

  3. (Opsiyonel) /analyze endpoint'ine use_cache parametresi:
     ```python
     def analyze(image: UploadFile = File(...), use_cache: bool = Form(True)):
         ...
         result = agent.analyze(tmp_path, use_cache=use_cache)
     ```

  4. Çalıştırma: `uvicorn api:app --reload`
     Tarayıcıda http://127.0.0.1:8000/ açılır, index.html gelir.
